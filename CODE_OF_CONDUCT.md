# Code of Conduct

## Our Pledge
We pledge to make participation in our community a harassment-free experience.

## Our Standards
Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints

## Enforcement Responsibilities
The maintainers are responsible for enforcing this Code of Conduct.

## Scope
This code applies both within project spaces and in public spaces when an individual is representing the project.

## Enforcement
Instances of abusive, harassing, or otherwise unacceptable behavior may be reported.

## Enforcement Guidelines
Reports will be reviewed and investigated.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
